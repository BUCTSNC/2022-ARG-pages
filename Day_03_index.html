As the 5*5 matrix shows , if you obtain the plain string "ordbib" , you are supposed to follow the rules:

* First , you should divide the plain string "oribdb" into groups whose size is 2 . Now we can get three groups "or","ib","db".

* From the 5*5 matrix , character 'o' and character 'r' are in the same row . You should move one unit right(by loop) and then get substring "co".

* Character 'd' and character 'b' are in the same column, You should move one unit down(by loop) and then get substring "bg".

* Character 'i' and character 'b' are not in the same row or column . So you should find a particular rectangle whose vertices contains both 'i' and 'b' , find another two vertices in some order and then you get substring "de".

Finally , merge all substrings and then get "cobgde".

矩阵的大致构成是这样的：
